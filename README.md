# HoneyHeist
Digital Prototype made in Unity.

## Controls:
Move the player with the arrow keys.
Rotate the stage counter clockwise with "A" and clockwise with "D".

Press "R" to reset the level.

## Download Instructions:
### Windows:
Download the zip file, extract it, and open the executable ThrowAway_Prototype.exe.
**All files that were in the zip file must be in the same directory as the executable**
